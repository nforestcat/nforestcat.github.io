운영체제와 리눅스 기초 정리

운영체제와 리눅스 기초 수업을 들으면서 정리한 내용이다. 전체 흐름은 다음과 같다.

운영체제 소개
→ 프로세스
→ 메모리
→ 파일 시스템
→ 리눅스 입문
→ 리눅스 파일 시스템
→ 리눅스 명령어
→ 프로세스와 네트워크
